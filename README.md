# Blood-Management-System
A Java-based blood management system developed as a college group project.

## Features
- Blood type compatibility check
- Hospital and donor management
- User management system
- Data storage and retrieval

## Tech Stack
- Java

## How to run
1. Clone the repository
2. Open in your Java IDE
3. Run BloodManagementSystem.java
